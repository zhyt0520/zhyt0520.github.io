Win11 关闭任务栏缩略图

发表于 2025-12-25 分类于 软件 Waline：

系统版本 Windows 11 25H2

  1. 下载 ViveTool，地址 https://github.com/thebookisclosed/ViVe/releases

  2. 使用下载的工具，命令行运行
    .\vivetool /disable /id:29532725

  3. 修改注册表
    HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Taskband
    新建 DWORD (32位) 值，命名为 NumThumbnails，数值设置为 0

  4. 重启系统
