近来可好

发表于 2017-08-27 更新于 2021-12-30 分类于 文字 Waline：

迷雾茫茫，何其寻探。随心随性，一路玩赏；迎难而上，披荆斩棘。责为趣，苦寻乐，魂魄不分，形神如一。

窗前

坐在电脑前，迎着窗子，清风徐来，初秋的凉爽没有丝毫的吝啬之意。外面路边一盏路灯，直矗矗的站着，坚定却又木讷。他将头顶一片光芒用力向脚下洒去，打在了树梢的绿叶上，溅在了脚下的柏油路，光影之间，也许真的藏着另一个次元。蟋蟀的声音窸窸窣窣，在两楼间不知反射几次，明亮、清澈，径直闯来。细细而听，又仿佛在房间的角落，自顾自讲述着不知名的故事。

我不禁嘴角微扬，转念，心中却是空白。

路上一辆白色的车子缓缓驶过，不知道他从哪来，不知道他到哪去。只是凑巧，我在楼上目送，愿所有的人平安，愿这个世界和平。

盛夏之后的清凉虽爽，却是不得多享。孩子睡了，一则怕凉，二则怕吵，只好关窗闭户。

外面依然清亮，只是我已听不到你。

妞儿

女儿长的很快。仿佛刚会翻身还是昨日，今时已经可以爬的飞快。

一天天的陪伴，看她哭，看她笑，有满心欢喜，有生气烦躁，也有哭笑不得。

已经八个多月，还是十八斤，近来体重增速很低。

想想相机又在某个角落吃了不少的灰了。不该不该。

只盼她健康成长。

CODING

断断续续的，有过连续两三天一个字没动，也有过从入夜开始一直写到天蒙亮。昨天突然发现，最大的一个文件，竟然已经有了八万多个字符。

这两天又看到那个说法，“七年就是一辈子 ”。毕业已六年，我也快到下一个辈子了吗？也许，不知道，至少现在，还没准备好。

纠结了许久是否要换方向，近来在心里逐渐明晰，也算终于做出了决定。只是时机的把握和事情的运作仍然会有未知的变数，会变成康庄坦途，还是会需要披荆斩棘，晓不得。

只管向前。

事事儿

家家有本难念的经。

与生俱来的，接受；自己选择的，承受；无法改变的，忍受。

当然，能做的，还有很多很多。生活，尽心而做，尽力而为，相信着，一切都会好起来的。

胖

贪吃、熬夜、少运动。

控制饮食，尽量早睡，定期锻炼。

“花了一生的时间，终于，把自己变成了原本讨厌的样子”。

之前想买个 kindle 的，纠结了一阵。现在决心不买了，怕弄回来之后它会向我那块 raspberry pi 一样，躺灰。但是书还是要读的，现在脑袋已经跟木头一样，再下去就变成朽木了。先把手上这几本看完。
