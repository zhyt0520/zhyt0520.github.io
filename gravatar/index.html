Gravatar全球通用头像

发表于 2013-05-22 更新于 2021-12-30 分类于 软件 Valine：

Gravatar 是 Globally Recognized Avatar 的缩写，可以翻译成“全球通用头像”。这是 gravatar.com 网站提供的一项服务，它可以把注册用户上传的头像图片根据用户的邮箱信息显示到其他网站上。这个东东首先是在国外独立 wordpress 博客上流行的，只要在 gravatar 上注册并设置自己的头像，然后在支持 gravatar 的网站留言时就可以自动显示用户在 gravatar 上设置的头像。

当然我的 一语 就是用 wordpress 搭建的，如果你想在这里留言的时候显示个自己的头像，那么马上到 gravatar.com 上面注册吧。输入 gravatar.com 的话会自动跳转为 en.gravatar.com ，英文的，如果非得想看到几个汉字的话，那网址就输 cn.gravatar.com 吧。

貌似当年我注册的那时候可以直接在 gravatar 上注册的，现在的话必须用 wordpress 账户才行（好麻烦啊……）。那么如果你有 wordpress 账户的话，直接点右上角的 sign in 登录，如果没有，需要先转到 wordpress.org 注册。第一次登录的时候貌似还需要确认 gravatar.com 对你 wordpress 账户的访问权限，同意即可。（如果这一步网页刷不出来的话，请找国外代理，什么在线代理、vpn、ssh 或者 翻greatwall 的小软件均可，过一段时间我肯定也写一篇关于 伟大firewall 的文章，会发在“软件”分类下，地址 http://aword.net/software）

登录好之后就简单了，你可以设置多个邮箱，其中一个是 Primary address 主邮箱，还可以上传多个图片，可以给不同的邮箱选择不同的图片头像。好了，如果需要的话，这功能还是挺有用的。（zhyt习惯所有地方全用一个头像）

这是我的账户界面。

O.K. 全部设置好了再回 一语 看看你的留言头像吧。记得只要输入自己的邮箱，你这个头像在所有支持 gravatar 的网站上都可以显示的哦。
